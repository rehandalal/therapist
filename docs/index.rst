Therapist
=========

Therapist is a tool to help create robust, easily-configurable
`pre-commit hooks`_ for `Git`_. This allows you to find and fix common issues
before you actually commit code to your repo.

.. _pre-commit hooks: https://git-scm.com/book/en/v2/Customizing-Git-Git-Hooks
.. _Git: https://git-scm.com/
